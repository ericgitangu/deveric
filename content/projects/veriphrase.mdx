A full-stack confirmation code encoder and decoder application designed for users and customer care representatives to securely generate, verify, and decode confirmation codes.

## Built with

- Next.js
- TypeScript
- Python (FastAPI)
- Docker
- CSS/Tailwind

## Key Features

### Code Generation
- Secure code encoding algorithms
- Customizable code formats
- Time-based expiration

### Verification Workflow
- User-friendly verification interface
- Customer support dashboard
- Real-time code validation

### Security
- Encrypted code storage
- Rate limiting
- Audit logging

## Tech Stack

### Frontend
- Next.js with TypeScript
- Modern CSS styling
- Responsive design

### Backend
- Python FastAPI
- RESTful API design
- Docker containerization

### Infrastructure
- Docker for deployment
- Environment-based configuration
- Scalable architecture

## Use Cases

- Customer support verification
- Account recovery workflows
- Secure transaction confirmations
- Multi-factor authentication support
