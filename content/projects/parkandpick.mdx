A production e-commerce storefront built on Next.js Commerce with Shopify integration, featuring seamless checkout, inventory management, and optimized performance.

## Built with

- Next.js 14
- Shopify Storefront API
- Tailwind CSS
- Vercel Edge Functions
- TypeScript

## Key Features

### Modern Storefront
- Server-rendered product pages
- Real-time inventory sync
- Fast checkout flow
- Mobile-first design

### Performance Optimized
- Edge-rendered pages
- Image optimization
- Incremental static regeneration
- Core Web Vitals optimized

### E-commerce Essentials
- Cart management
- Wishlist functionality
- Order tracking
- Search with filters

## Tech Stack

### Frontend
- Next.js 14 with App Router
- React Server Components
- Tailwind CSS
- TypeScript

### Backend
- Shopify Storefront API
- GraphQL queries
- Vercel Edge Functions

### Infrastructure
- Vercel hosting
- Custom domain
- Edge caching
- Analytics
