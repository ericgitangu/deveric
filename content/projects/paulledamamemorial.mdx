# Description

A memorial website honoring the life and legacy of Paul Ledama Sekento, built with modern web technologies and deployed on Fly.io.

## Features

- Next.js 15 with App Router
- Tailwind CSS v4 for styling
- Dark/Light theme support (defaults to dark)
- Responsive design for all devices
- PWA support with offline capabilities
- SEO optimized with Open Graph tags
- Biography, tributes, gallery, hymns, and programme sections
- Elegant typography with Playfair Display and Inter fonts

## Tech Stack

- **Framework:** Next.js 15
- **Styling:** Tailwind CSS v4
- **Theme:** next-themes
- **Deployment:** Fly.io
- **PWA:** next-pwa
- **TypeScript:** Full type safety

## Purpose

This memorial website serves as a digital tribute to Paul Ledama Sekento, providing family and friends a place to:

- Read his biography and life story
- Share and view tributes from loved ones
- Browse photos from his life
- Access funeral service programme details
- Find hymns for the celebration of life
